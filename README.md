# Kisan Assistant (Phase 1)

A farmer-friendly web application designed to help Indian farmers with crop problems, information, and community interaction.

## Features
* **Language Support:** English, Hindi, Marathi, Kannada, Telugu.
* **Crop Doctor (Mock AI):** Diagnoses simple issues based on keywords (e.g., "yellow", "holes").
* **Community:** Farmers can post tips and YouTube video links.
* **Market Prices:** View mock prices for Pune and Mumbai markets.
* **Schemes:** Information on government schemes.

## Tech Stack
* Python (Flask)
* SQLite (Database)
* HTML/CSS (No external frameworks)

## How to Run (GitHub Codespaces)
1.  Open this repository in GitHub Codespaces.
2.  In the terminal, run: `pip install flask`
3.  Start the app: `python app.py`
4.  Click "Open in Browser" when the notification appears.
